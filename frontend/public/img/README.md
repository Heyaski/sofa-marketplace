# Изображения для VizHub.art

Эта папка содержит все изображения для фронтенда маркетплейса.

## Текущие изображения:

### 1. **hero-image.jpg** ✅

- **Описание**: 3D модель дивана для геройской секции
- **Размер**: 600x600px или больше
- **Формат**: JPG
- **Использование**: Главная страница, геройский блок

### 2. **sofa-card.svg** ✅

- **Описание**: Изображение дивана для карточек товаров
- **Размер**: 300x300px или больше
- **Формат**: SVG
- **Использование**: Карточки товаров в каталоге

### 3. **Дополнительные иконки** ✅

- **logo.svg** - логотип VizHub.art (используется в Header и Footer)
- **menu-burger.svg** - иконка меню (используется в кнопке "Каталог")
- **inst.svg** - Instagram (используется в Footer)
- **telegram.svg** - Telegram (используется в Footer)
- **whatsapp.svg** - WhatsApp (используется в Footer)
- **youtube.svg** - YouTube (используется в Footer)
- **button-\*.svg** - иконки кнопок (готовы к использованию)

## Как использовать:

1. Изображения уже находятся в папке `frontend/public/img/`
2. Они автоматически доступны по путям:
   - `/img/hero-image.jpg`
   - `/img/sofa-card.svg`
   - `/img/logo.svg`
   - и т.д.

## Рекомендации:

- Используйте оптимизированные изображения для быстрой загрузки
- Поддерживайте соотношение сторон 1:1 для карточек
- Используйте качественные 3D рендеры для лучшего восприятия
